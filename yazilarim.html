Git ve GitHub Nedir?

Git, bir sürüm kontrol sistemidir. Yani yaptığın değişiklikleri takip eder. Kod yazarken geçmişe dönmeni sağlar.

GitHub ise bu Git ile çalıştığın projeleri internete yüklemeni sağlar. GitHub, projelerini başkalarıyla paylaşmanı ve saklamanı kolaylaştırır.

Aralarındaki Fark

  • Git bir araçtır. Bilgisayarında çalışır.
  • GitHub bir web sitesidir. Git ile yaptığın işleri internette saklar.

En Çok Kullanılan Git Komutları

  • git init: Yeni bir Git projesi başlatır.
  • git add .: Tüm değişiklikleri ekler.
  • git commit -m "açıklama": Yaptığın değişiklikleri kaydeder.
  • git push: GitHub’a yükler.
